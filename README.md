App Description

The Employee Contact Manager App is an Android application designed to store and manage employee contact details in an organized manner. It allows users to register employees 
by entering name, ID, phone number, and department using interactive input fields and a Spinner. The application displays employee profiles in a RecyclerView with CardView 
layout, enabling users to select an employee and perform quick communication actions such as calling, sending SMS, or emailing after confirmation through a DialogBox. This app 
demonstrates practical implementation of list display, activity navigation using Intents, and real-time communication features.
